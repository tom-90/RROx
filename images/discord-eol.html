_tom() 15-02-2024Hi everyone,

I have some sad news about RROx I want to share with you guys today. After giving it a lot of thought, I've decided to stop supporting the RROx mod. This was an incredibly tough decision for me to make, as I know there are still many community members that really like using RROx and couldn't play the game without it. However because of the lack of official modding support from the game itself, and the incredible complexity that RROx has to go through to be functional, I feel like it is the right time to make this decision, as I don't think I would be able to find the motivation to keep RROx supported if another major breaking update for the game were to release.
Let me summarize what will happen to RROx in the short term:
  • RROx will remain fully functional in it's current state for the current version of the game, however no bug fixes will be made and no new content or features will be added. If a new version of the game releases, RROx could stop working either partially or fully depending on the size of the updates.
  • RROx will remain downloadable from https://tom-90.github.io/RROx (with a warning that it is unsupported).
  • The server that is behind RROx's plugin download system and also it's multiplayer session sharing system will remain online for the forseeable future (I will do my best to keep it up for another year or two).
  • The source code behind RROx will remain available on GitHub, but the repository will be marked as read-only.
  • This Discord server will be archived and changed into read-only mode. This will happen on the 17th of February. If you would like to keep discussing about the game itself, I would like to refer you to the official Railroads Online discord or to the Blueberry Valley Community, which is a community broader than just Railroads Online.

All in all I would like to thank everyone who is or has been a part of the RROx community. The last two years have been incredible and beyond my wildest expectations. I would have never thought that a silly little hobby project of mine would grow into a fully flegded game mod that has been used and appreciated by so many people. I also want to thank the mods and maintainers in particular, for both their contributions to the RROx code, as well as for their help in moderating this Discord server.

Thank you for the incredible support. It has been an amazing journey.

~Tom
